SIL's Yupik Cyrillic (Russian based) keyboard

русский

This Yupik Cyrillic keyboard (sil_yupik_cyrillic_ru) is based on a standard Russian layout. It has been modified as described below, to give access to the additional characters required for the Yupik language.

Desktop Keyboard Layout

Modifications to the Russian layout

Yellow keys—press any of these keys twice to generate the corresponding consonant with descender: қ ң ӷ ҳ. Pressing 'в' twice will generate 'ў'.

Orange keys—press any of these keys twice to generate the corresponding vowel with macron above it: ӯ а̄ я̄ ӣ ю̄

Default (unshifted)

Shift

Right ALT (unshifted)
